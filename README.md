# 💰 AI Expense Tracker

A modern AI-powered personal finance management application built with **React**, **Zustand**, **Tailwind CSS**, and **Recharts**. It helps users manage income, expenses, budgets, accounts, and financial analytics through an intuitive dashboard with an AI-powered assistant.

---

## ✨ Features

### 📊 Dashboard
- Financial summary cards
- Total Income
- Total Expenses
- Current Balance
- Cash & Bank Balance
- Monthly Expense Chart
- Category-wise Expense Chart

### 💰 Income Management
- Add Income
- Edit Income
- Delete Income
- Recurring Income Support

### 💸 Expense Management
- Add Expenses
- Edit Expenses
- Delete Expenses
- Recurring Expense Support

### 📋 Transactions
- View All Transactions
- Search Transactions
- Filter by Month
- Filter by Date Range
- Sort by Date
- Sort by Amount
- View Transaction Details
- Edit Transactions
- Delete Transactions
- Export Transactions to CSV

### 📂 Categories
- Add Categories
- Delete Categories
- Dynamic Category Management

### 🎯 Budget Planner
- Create Monthly Budgets
- Edit Budget
- Delete Budget
- Budget Progress Tracking
- Spending Percentage
- Budget Alerts

### 🏦 Account Management
- Cash Account
- Bank Account
- Custom Accounts
- Opening Balance
- Transfer Between Accounts

### 📈 Analytics
- Income Analytics
- Expense Analytics
- Savings Overview
- Monthly Trends
- Category-wise Spending
- Transaction Statistics

### 🤖 AI Expense Assistant
Ask questions in natural language, such as:

- What is my total income?
- What is my total expense?
- What is my current balance?
- Show my highest expense.
- How much did I spend on Food?
- Analyze my spending.

The AI assistant is designed to support future integration with OpenAI, Gemini, Claude, or local LLMs.

# 🚀 Getting Started

## Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/expense-tracker.git
```

## Navigate to the project

```bash
cd expense-tracker
```

## Install dependencies

```bash
npm install
```

## Start the development server

```bash
npm run dev
```

## Build for production

```bash
npm run build
```
